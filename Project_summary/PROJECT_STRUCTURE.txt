WORKFLOW DIAGRAM
================

Setup Phase:
  1. pip install -r requirements.txt
  2. Install Ollama and pull a model (mistral/llama3.2/qwen2.5)
  3. copy .env.example .env (edit with Ollama config)
  4. python test_setup.py (verify setup)
  5. python data_ingestion.py (load data)

Runtime Phase:
  5. python main.py (start agent)
  6. Ask questions in natural language
  7. Agent queries MongoDB and responds
