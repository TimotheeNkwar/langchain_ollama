WORKFLOW DIAGRAM
================

Setup Phase:
  1. Install Ollama from ollama.ai
  2. Pull a model: ollama pull mistral (or llama3.2/llama3.1/qwen2.5)
  3. pip install -r requirements.txt
  4. copy .env.example .env (edit with Ollama config)
  5. python testing/test_setup.py (verify setup)
  6. python data_ingestion.py (load data)

Runtime Phase:
  7a. python main.py (start CLI agent)
  7b. uvicorn api:app --reload (start REST API on port 8000)
  8. Ask questions in natural language
  9. Agent queries MongoDB and responds
  10. Access API docs at http://localhost:8000/docs
